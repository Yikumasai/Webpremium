## 📖 소개

Webpremium은 스마트 프리로드 기술을 통해 **제로 지연** 웹 브라우징 경험을 구현하는 혁신적인 Chrome 확장 프로그램입니다. 링크에 마우스를 올리면 백그라운드 프리로드 창에서 페이지를 미리 엽니다. 실제로 링크를 클릭하면 프리로드된 탭이 메인 창으로 매끄럽게 이동하여 대기 시간을 느끼지 못하게 합니다.

### ✨ 핵심 특징

- 🎯 **제로 지연 경험** - 마우스 호버로 프리로드, 클릭으로 즉시 열기
- 🪟 **프리로드 창 기술** - 독립 창에서 프리로드, 메인 창에 간섭 없음
- 🔄 **스마트 탭 관리** - 이미 열린 탭을 자동으로 감지하고 이동
- 📊 **실시간 통계** - 프리로드 효과와 절약된 시간 추적
- 🎨 **현대적인 인터페이스** - 다크 모드 지원, 깔끔하고 아름다운 인터페이스
- ⚙️ **높은 커스터마이징** - 풍부한 설정 옵션으로 개인화된 요구 충족

---

## 🎯 기능

### 핵심 기능

#### 1. 스마트 프리로드
- **마우스 호버 트리거** - 링크에 마우스를 올리면 자동으로 프리로드
- **조정 가능한 지연 시간** - 0-1000ms의 호버 지연 설정 지원
- **근처 링크 예측** - 마우스 근처의 링크를 스마트하게 식별하고 프리로드
- **프리로드 수량 제어** - 최대 동시 프리로드 수 설정 가능 (1-10개)

#### 2. 프리로드 모드
- **프리로드 창 모드 (권장)** - 독립된 최소화 창에서 프리로드, 완전한 페이지 로딩, 메인 창에 간섭 없음
- **iframe 프리로드 모드** - 경량 프리로드 방식, 호환성 우수

#### 3. 스마트 탭 관리
- **중복 탭 감지** - 동일한 URL의 탭을 자동으로 감지
- **자동 이동** - 이미 열린 링크를 클릭하면 기존 탭으로 자동 이동
- **매끄러운 이동** - 프리로드 탭이 메인 창으로 매끄럽게 이동
- **메모리 최적화** - 중복 탭 감소, 메모리 사용량 감소

#### 4. 네트워크 인식
- **스마트 감지** - 네트워크 상태를 자동으로 감지
- **적응형 전략** - 느린 네트워크에서 자동으로 프리로드 감소
- **데이터 절약** - 약한 네트워크 환경에서 데이터 낭비 방지

#### 5. 시각적 표시기
- **프리로드 상태 표시** - 링크 옆에 작은 점으로 프리로드 상태 표시
- **로딩 애니메이션** - 주황색 점은 로딩 중을 나타냄
- **로딩 완료 표시** - 녹색 점은 프리로드 완료를 나타냄

#### 6. 사이트 규칙 관리
- **사용자 정의 규칙** - 특정 사이트에 대해 프리로드 활성화 또는 비활성화
- **도메인 수준 제어** - 도메인별 정밀한 프리로드 제어
- **우클릭 메뉴** - 현재 사이트의 프리로드 상태를 빠르게 전환

#### 7. 통계 및 분석
- **프리로드 횟수** - 총 프리로드 횟수 기록
- **적중률 통계** - 프리로드의 효과적인 활용률 계산
- **시간 절약** - 절약된 총 시간 통계
- **세션 시간** - 현재 세션의 사용 시간 표시

### 단축키

- `Alt + P` - 프리로드 기능 빠르게 켜기/끄기
- `Alt + C` - 모든 프리로드 캐시 지우기

### 우클릭 메뉴

- **이 링크 프리로드** - 선택한 링크를 수동으로 프리로드
- **이 사이트에서 프리로드 활성화/비활성화** - 현재 사이트의 프리로드 상태를 빠르게 전환

## 📦 설치 방법

### 방법 1: 개발자 모드로 설치

1. **소스 코드 다운로드**
   [release](https://github.com/Yikumasai/Webpremium/releases) 페이지에서 다운로드
   
   또는
   
   ```bash
   git clone https://github.com/Yikumasai/webpremium.git
   ```

2. **Chrome 확장 프로그램 페이지 열기**
   - 주소창에 `chrome://extensions/` 입력
   - 또는 메뉴 → 도구 더보기 → 확장 프로그램

3. **개발자 모드 활성화**
   - 페이지 오른쪽 상단의 "개발자 모드" 스위치 켜기

4. **확장 프로그램 로드**
   - "압축해제된 확장 프로그램을 로드합니다" 클릭
   - 다운로드한 `webpremium` 폴더 선택

5. **설치 완료**
   - 확장 프로그램 아이콘이 브라우저 도구 모음에 나타남
   - 아이콘을 클릭하여 설정 패널 열기

### 방법 2: Chrome 웹 스토어
> 준비 중

## 🎮 사용 방법

### 기본 사용법

1. **확장 프로그램 활성화**
   - 설치 후 확장 프로그램은 기본적으로 활성화됨
   - 도구 모음 아이콘을 클릭하여 상태 확인

2. **프리로드 경험**
   - 임의의 링크에 마우스를 올림
   - 설정된 지연 시간 대기 (기본값 100ms)
   - 링크 옆에 녹색 점이 나타나면 프리로드 완료
   - 링크를 클릭하면 즉시 열림

3. **통계 확인**
   - 확장 프로그램 아이콘 클릭
   - "통계" 탭으로 전환
   - 프리로드 효과와 절약된 시간 확인

### 고급 설정

#### 호버 지연 조정
- 설정 패널 열기
- "호버 지연" 슬라이더 드래그
- 권장값: 100-300ms

#### 프리로드 수 설정
- 설정 패널 열기
- "최대 프리로드 수" 슬라이더 드래그
- 권장값: 3-5개

#### 프리로드 모드 선택
- **프리로드 창 모드**: 완전한 프리로드, 최고의 경험 (권장)
- **iframe 모드**: 경량, 호환성 우수

#### 사이트 규칙 관리
1. "사이트 규칙" 탭으로 전환
2. "규칙 추가" 버튼 클릭
3. 도메인 입력 (예: example.com)
4. 활성화 또는 비활성화 상태 설정

## ⚙️ 작동 원리

### 프리로드 흐름

```
사용자가 링크에 호버
    ↓
지연 시간 대기
    ↓
네트워크 상태 확인
    ↓
사이트 규칙 확인
    ↓
프리로드 창 생성
    ↓
프리로드 창에서 탭 열기
    ↓
프리로드 창 최소화
    ↓
사용자가 링크 클릭
    ↓
탭을 메인 창으로 이동
    ↓
탭 활성화
    ↓
완료!
```

### 기술 아키텍처

- **Content Script** - 페이지 링크 이벤트 감지, 프리로드 트리거
- **Background Service Worker** - 프리로드 창과 탭 관리
- **Popup UI** - 설정 인터페이스와 통계 정보 제공
- **Chrome Storage API** - 설정과 통계 데이터 영구 저장

### 프리로드 창 기술

확장 프로그램은 독립된 프리로드 창을 사용하여 페이지를 프리로드합니다:

1. 작은 normal 타입의 창 생성
2. 즉시 해당 창 최소화
3. 창 내에서 프리로드 탭 생성
4. 사용자가 클릭하면 탭을 메인 창으로 이동
5. 탭을 활성화하고 메인 창에 포커스

이 방식의 장점:
- ✅ 페이지 완전 프리로드 (JavaScript, CSS, 이미지 등 포함)
- ✅ 메인 창에 전혀 영향 없음
- ✅ 탭을 매끄럽게 이동 가능
- ✅ 모든 사이트와 복잡한 페이지 지원

## ❓ 자주 묻는 질문

### Q: 프리로드가 많은 데이터를 소비하나요?
A: 확장 프로그램은 네트워크 상태를 스마트하게 감지하고 느린 네트워크에서 자동으로 프리로드를 줄입니다. "최대 프리로드 수"를 조정하여 데이터 소비를 제어할 수도 있습니다.

### Q: 프리로드가 브라우저 성능에 영향을 주나요?
A: 프리로드는 독립된 창을 사용하므로 메인 창 성능에 미치는 영향이 최소화됩니다. 또한 확장 프로그램은 만료된 프리로드 콘텐츠를 자동으로 정리합니다.

### Q: 일부 사이트에서 프리로드가 실패하는 이유는?
A: 일부 사이트에는 보호 메커니즘이 있을 수 있습니다. "사이트 규칙"에서 이러한 사이트의 프리로드를 비활성화할 수 있습니다.

### Q: 링크가 프리로드되었는지 어떻게 알 수 있나요?
A: "표시기 표시"를 활성화하면 프리로드 완료된 링크 옆에 녹색 점이 표시됩니다.

### Q: 프리로드 창이 표시되나요?
A: 아니요. 프리로드 창은 즉시 최소화되어 브라우징 경험에 전혀 영향을 주지 않습니다.

### Q: 특정 사이트에 대해 프리로드를 비활성화할 수 있나요?
A: 예. "사이트 규칙" 탭에서 도메인 규칙을 추가하거나 페이지를 우클릭하여 "이 사이트에서 프리로드 활성화/비활성화"를 선택하세요.
